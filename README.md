# Generative AI Research Project

This repository stores materials for the research project "Using Generative AI Tools - Boon or Bane". It is organised to support file management, version control, and collaboration using GitHub.

## Project Structure

- `Literature_Review/`: journal articles, reference papers, and literature sources
- `Quantitative_Analysis/`: survey data, survey questions, and analysis scripts
- `Qualitative_Analysis/`: interview transcripts and other qualitative research materials
- `Drafts_and_Reports/`: proposal drafts, paper drafts, and submitted reports
- `Additional_Materials/`: images and supporting project materials

## How To Navigate

Open each top-level folder to access the relevant research materials. Files are grouped by research activity so collaborators can quickly find literature, datasets, transcripts, reports, and supporting media.

## Collaboration Guidelines

- Use clear and consistent file names
- Commit changes with meaningful commit messages
- Review files before uploading to keep folders organised
- Use branches and pull requests when working as a team

## Notes

Some research materials, especially interview-related files, may require restricted access and careful handling to protect privacy and research integrity.
